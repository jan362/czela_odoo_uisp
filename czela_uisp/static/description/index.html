CZELA UISP Integration

Synchronize UISP network devices with Odoo

Features

Device Synchronization

Automatically sync network devices from UISP to Odoo every 15 minutes. View device status, metrics (CPU, RAM, signal), and GPS locations.

ČTÚ Compliance

Generate ČTÚ ART252 export reports for FTTH, WiFi, and FWA technologies. Automatic technology classification based on device models.

Key Capabilities

  • Real-time device status monitoring
  • Automatic sync via cron jobs
  • MAC address-based device matching
  • Technology classification (FTTH/WiFi/FWA)
  • Manual sync wizard
  • ČTÚ ART252 export wizard
  • Site hierarchy management
  • Partner integration with smart buttons

Configuration

Configure UISP connection in Settings → Technical → System Parameters:

  • uisp.base_url - UISP server URL (e.g., https://10.93.9.8)
  • uisp.api_key - UISP API authentication key
  • uisp.verify_ssl - SSL certificate verification (true/false)
